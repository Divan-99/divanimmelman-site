Modular OCR System

Configurable Python platform that extracts document numbers and barcodes from PDFs automatically. 002 ● Production

Iscar Label Printing App

Django web app deployed on the factory network at Iscar South Africa. Staff search a product code, see a live label preview (code, description, bin location, barcode), and hit Print — ZPL is sent directly to the Honeywell PD45 network printer over TCP port 9100 with no browser dialog. Product data is read from Excel and cached in memory.

PythonDjangoZPL PandasOpenPyXLTCP/IP

003 ● Production

Iscar Document Portal

Internal HTTPS portal serving delivery notes, purchase orders, picking slips, and shipments from network drives. Built with Django and Waitress on Windows, reverse-proxied through Nginx on Ubuntu with SSL termination. Features live AJAX search, path-traversal-safe downloads, and environment-based config — deployed at isdoc.sa.iscar.com on the corporate LAN.

PythonDjangoNginx WaitressLinuxHTTPS

Process Control Dashboard

Django dashboard for monitoring and controlling all internal Django apps and OCR scripts from a single browser tab. Polls process status every 5 seconds, streams live stdout logs, and exposes a REST API to start, stop, and restart any registered process — with login-protected access and Home Assistant integration for real-time status visibility.
